# figma-fetch

Fetch Figma File artboard as JSON

### Install

```
npm install
```

### Create `.env`

Add Figma API Key and Figma ID (artboard URL) to `.env` file. E.g. https://www.figma.com/file/<FIGMA_ID>/

```
API_KEY=********
FIGMA_ID=*******
```

### Run

```
npm run start
```

Fetches Figma JSON and saves to file
