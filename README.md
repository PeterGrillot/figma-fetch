# figma-fetch

Fetch Figma File artboard as JSON

### Install

```
npm install
```

### Create `.env`

Add Figma API Key and Figma ID (artboard URL) to `.env` file. E.g. https://www.figma.com/file/<THIS_ID>/

```
API_KEY=********
FIGMA_ID=*******
```

### Run

```
npm run start
```

Fetches JSON and saves file
